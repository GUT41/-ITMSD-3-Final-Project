# LexiSearch — Offline Dictionary App

ITMSD 3 Capstone Project: A mobile dictionary app with instant autocomplete (Trie) and fuzzy search (Levenshtein distance).

## Tech Stack

- **Framework:** React Native + Expo SDK 52
- **Navigation:** Expo Router (tabs, stack, modal)
- **State:** Zustand
- **Storage:** AsyncStorage + JSON
- **Testing:** Jest

## Getting Started

```bash
npm install
npx expo start
```
